# SkillShub - 云端技能平台开发计划

**项目名称：** SkillShub（原 Local ClawHub）
**目标：** 构建云端技能管理平台，支持局域网和公网访问

**制定日期：** 2026-03-06

---

## 📋 需求确认

### 核心需求

1. **云端化**
   - 技能集中托管（云数据库 + 文件存储）
   - 版本管理（多个版本、latest 标签）
   - 下载统计和排行榜

2. **双模式访问**
   - 🏢 局域网模式（内部人员）
     - 完整功能：浏览、搜索、上传、下载、评论
     - 内部用户可发布技能
     - 无需外部访问
   - 🌐 公网模式（公司人员）
     - 浏览技能、搜索技能
     - 下载技能
     - 登录后可评论

3. **多用户支持**
   - 用户注册/登录
   - 权限管理（所有者、管理员、管理员）
   - 团队协作（技能所有权转移）

4. **社区功能**
   - 技能评论
   - 技能收藏/点赞
   - 趋势榜和推荐

## 📦 技术选型

### 完整云端平台（自建）

| 组件 | 技术选型 | 理由 |
|--------|-----------|------|
| **后端** | **Rust + Actix/Axum** | 高性能、内存安全、跨平台 |
| **前端** | **React + Vite + shadcn/ui** | 现代化、组件丰富、易维护 |
| **数据库** | **PostgreSQL 15** | 成熟、ACID、JSON 支持 |
| **文件存储** | **本地文件系统** | 完全控制、成本可控 |
| **向量搜索** | **Qdrant + fastembed-rs** | 轻量级、本地向量、无需 OpenAI |
| **CLI 工具** | **TypeScript + oclif** | 跨平台、易维护 |
| **部署** | **Docker + Docker Compose** | 容器化、易扩展 |
| **反向代理** | **Nginx** | HTTPS、负载均衡 |
