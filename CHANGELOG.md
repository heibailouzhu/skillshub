## [0.1.0] - 2026-03-09

### Added / 新增
- SkillShub platform initialization / SkillShub 平台初始化
- Backend API with Rust + Axum / 基于 Rust + Axum 的后端 API
- Frontend UI with React + Vite / 基于 React + Vite 的前端界面
- PostgreSQL database integration / PostgreSQL 数据库集成
- Docker deployment configuration / Docker 部署配置
- User registration and login / 用户注册和登录
- Skill upload and management / 技能上传和管理
- Search functionality / 搜索功能
- Comment system / 评论系统
- Favorite and rating system / 收藏和评分系统

### Tech Stack / 技术栈
- Backend: Rust 1.94, Axum 0.7, SQLx 0.7
- Frontend: React 18, Vite 5, Tailwind CSS
- Database: PostgreSQL 15
- Deployment: Docker, Docker Compose, Nginx
